# 更新日志

旺财 / Larkwing 的版本说明。版本号遵循语义化版本(SemVer);每个版本一节,标题格式 `## 版本号 — 日期`。

> 发布时,GitHub Release 的正文由 CI **自动抽取「对应版本这一节」**填入(见 `.github/workflows/release.yml`)。
> 出新版 = 在下方最上面加一节 `## x.y.z — 日期`,把该版变化写在它下面即可,无需手动填 Release 正文。

## 0.2.20 — 2026-07-18

修一个真机撞上的事:让它「发文件到微信」,一直报发不出去。

### 修复

- 📤 **发文件到微信不再莫名失败** — 微信那头隔太久没跟它说话时,会话令牌会悄悄过期,之前它发文件 / 推提醒会一直失败还说不清原因(重试、重启渠道都救不回,只有你在微信上先发一句话才能好)。现在它撞上这种情况会自动去掉过期令牌重发一次,大多数时候自己就缓过来;实在发不出去也会把话说明白:「让 TA 在微信上先给我发一句话再试」。
- 🔍 **发没发出去,如实说** — 微信服务端某种形态的失败应答,此前会被当成「发成功了」、它误报「发出去了」;现在如实识别为失败并告诉你。

> ⚠️ **待真机确认(真微信)**:等下次自然撞上「隔了很久没聊」的场景让它发个文件——日志应有「疑似 context_token 过期,去掉令牌重发一次」一行、且文件真送达(真 iLink 接不接受无令牌发送只能这么验);提醒推回微信同样受益,顺带观察。

## 0.2.19 — 2026-07-15

它在网页上要点「付款 / 发布 / 删除」这类按钮时,会先停下来问你——电脑、悬浮窗、手机、开口说话,哪边点头都算;外加看图更聪明的一批改进。

### 新功能 / 改进

- ✅ **网页上的「大事」先问你** — 它替你操作网页时,翻页、搜索、填表、下载全程照旧不打扰;但要点到「确认支付」「立即购买」「发布」「删除」这类有真实后果的按钮,会先停下来弹一张确认卡:写清楚在哪个网站、要点哪个按钮(按钮文字是从页面上现抓的,不是它转述的),你点「继续」才真的点下去,点「先不要」或不理(一分钟过时)就不动。它自己拿不准某一步有没有后果时,也会主动来问。
- 📱 **不在电脑前也能点头** — 从手机(微信/钉钉/Telegram)让它办事撞上确认时,那台手机会收到一条「要在 XX 网站点『XX』,回『确认』就继续」;回别的话(比如「先别,改成到店自提」)= 不点,而且这句话直接接进对话让它改方案,事不作废。悬浮窗也能直接点(主窗收进托盘时靠它);语音喊它办事撞上确认,它会念出来问你,嘴上答「可以 / 算了」就行——听不清宁可当没听到,绝不冒点。
- 📒 **足迹页多了「确认过的操作」** — 每次确认留一行:点了什么、在哪个站、继续了还是没执行、是在电脑 / 悬浮窗 / 手机还是口头点的头。事后想查「上周它到底点没点」一眼就有。
- 👀 **它知道自己能不能看图了** — 用不带「眼睛」的模型(如 DeepSeek)时,截图、图片不再假装看过:传不过去会如实说明,不再一本正经描述它根本没看到的画面。新增「读图」能力:换了能看图的模型,让它「看看刚发的图 / 再看看上周那张」都行。
- 🧠 **模型「图片理解」能自己标了** — 大脑设置的高级区新增「图片理解」开关(自动 / 能看图 / 只看文字):自架的视觉模型能标上,目录说错了也能纠;内置目录同步校订(Kimi K2.5/2.6、通义 VL/Omni 家族已标为能看图)。
- 💬 **手机上回消息更顺了** — 家里一个人让它跑长差事(比如替你操作网页几分钟)时,另一个人发消息不再被卡着等,各聊各的互不挡道。

### 修复

- 修 Windows 版网页截图编译问题(0.2.18 的 CI 产物受影响,本版一并带上)。

> ⚠️ **待真机确认(Windows/真网/真麦)**:真购物或票据网站点到「支付/购买」→ 确认卡弹出、点「继续」真点下去、点「先不要」它如实收手不换路硬做;正常翻页/下载全程不被误问(误问多了反馈哪个词,删词即可);手机三渠道各验一遍「回『确认』继续 / 回别的改方案 / 不理超时」;语音喊它办事撞确认 → 念出来问 → 口头「可以 / 算了」有效、含糊时卡片还在;悬浮窗收托盘时能点;足迹页记录完整。

## 0.2.18 — 2026-07-14

它能在网页上「动手」了——填表单、点选项、需要时截图看画面;外加一批攒下来的改进:聊完主动收尾、家庭日记当天可见、手机聊天不断线、查资料续读长文、数据能从备份恢复。

### 新功能 / 改进

- 🖱️ **网页上能填表单、点选、截图了** — 以前它打开网页只能看和点,现在能往输入框填字、批量填整张表、选下拉、勾选、翻页、提交表单;说不清页面长啥样时还能截一张图看。发票、登记这类「开网页→填/下载」的差事能一步步做下来。遇到要登录/验证码/填密码,仍旧请你在那个可见小窗里自己操作、它接着干——密码这类敏感信息它绝不代填。
- 🗣️ **说「没事了」就收住** — 连续对话里说一句「没事了 / 先这样 / 拜拜」,它立刻安静回到待唤醒,不用干等几秒;重新喊名字随时还能叫醒。
- 📔 **家庭日记当天能看** — 以前要跨过一整天才补写,现在攒够了、趁空闲就动笔,今天的事当天就能看到。
- 💬 **手机上跟它聊不断线** — 单聊闲置久了自动开新话题、老对话留着;它到点提醒你时,你回一句就能接着那条老话题聊。删掉某个手机对话也不再永久报错。
- 🔎 **查资料能读长文、认下载链接** — 长文章能接着往下读、不再卡在半路;遇到 PDF 之类的直链会指路你直接下载再读,不把乱码当内容。
- 💾 **数据能从备份恢复 + 手机发的图能存电脑** — 换机 / 重装 / 盘坏后,拿之前导出的备份 zip 就能把数据(含克隆音色)恢复回来;手机发来的图片也能像文件一样「存到电脑 / 发给某某」了。

> ⚠️ **待真机确认(Windows)**:这一版都在 Mac 开发机验过、Windows 真机还没验。重点看——网页填表/提交/截图真能用(尤其截图在 Windows WebView2 上真抓得到图)、反爬网站填不进去时能顺畅交还给你操作;从备份 zip 恢复后数据完整、能开库;手机各渠道(微信/钉钉/Telegram)闲置轮换、提醒接得回、发图能存电脑。

## 0.2.17 — 2026-07-11

在手机上发文件给它,现在能真的「存到电脑」了;连发几个文件也不再被追问好几遍。

### 新功能 / 改进

- 📥 **手机发来的文件能存到电脑 / 整理了** — 以前在微信/钉钉发文件给它,它只会说「找不到文件、你重新发」——因为文件其实没落到电脑上。现在收到的文件会先存进本地收件区,它就能按你的话「存到桌面」「建个文件夹放进去」。扫描件(读不出文字的 PDF)也能存,存文件不需要读内容。
- 📎 **文件先攒着、等你说要做什么** — 手机上发文件不能同时打字(微信、钉钉都是文件单独一条)。以前每个文件一到就被当成一次请求,它没头没脑地瞎处理。现在发来的文件会先攒着、回你一句「文件收到了,想让我做什么?」,等你说了需求,再连同文件一起办。
- 🔕 **连发几个文件只提示一次** — 一口气发三个 PDF,它只在第一个时说一声「收到了」,后面的静默收下,不会追问你三遍。你发来说明后,它把这几个文件一起处理。

> ⚠️ **待真机确认**:微信/钉钉发多个文件 → 只提示一次 → 说「存桌面建文件夹」→ 文件真落到位;扫描件也能存;钉钉发文件(先文件后文字,隔十几秒)能凑成一次处理;Telegram 发文件带说明仍即时处理(不受攒批影响)。

## 0.2.16 — 2026-07-11

微信改成「一人一个机器人」:家里人各自扫码、同时在线,第二个人扫码不再顶掉第一个。

### 修复 / 改进

- 👨‍👩‍👧 **微信多绑定** — 真机实测发现 iLink 机器人是「一人一个」:第二个人扫码会把第一个人顶下线(旧的显示「无法连接」)。现在扫码 = 追加一个绑定:家里人各自扫自己的码、每人一个机器人、全部同时在线;同一个人重扫 = 换新不加行。设置页能看到绑定列表、可单独解绑。旧版绑过的自动迁移(建议重扫一次补齐身份)。
- 🏷️ **「允许的用户」改「额外允许」** — 绑定过的微信自动放行,扫码不再自动改这个名单(之前看起来像名单被顶换);一般留空即可。
- 📤 **出站互不串线** — 回复、提醒推送、发文件都按对话对应的绑定选机器人。

> ⚠️ **待真机确认**:两个微信各自扫码后**同时在线**收发(关键:确认腾讯侧允许一个客户端养多个 bot);解绑一路、另一路不受影响;提醒/发文件走对各自机器人;旧迁移绑定重扫后身份补齐。

## 0.2.15 — 2026-07-11

真机首验反馈一批修完:喊它立刻用人声答应、想说多久说多久;开机后「开着却叫不应」根治;文件能点名发微信;「跟妈妈说一声」不再认死名字。

### 新功能 / 改进

- 🗣️ **喊到立刻答应,录音不断** — 喊「逼踢」的瞬间它就出声应答(「嗯?」「在呢」…),不再是「叮」一声;你可以连着说(「逼踢帮我看看…」后半句不丢),说话中间想个词停顿一下也不再被切断(断句从 0.45 秒放宽到 0.8 秒、整段上限从 6 秒放宽到 12 秒)。应答音没准备好时(首次启动/断网)会退回「叮」,绝不会喊了没动静。
- 🏷️ **默认喊法只有「逼踢」** — BT 就是个英文名,起什么名字就怎么喊、不做特殊处理;「七二七四」退役(⚠️ 之前喊七二七四的这版起不再应)。
- 📲 **发文件能点名渠道** — 「发我微信」就走微信、「发我钉钉」就走钉钉;不点名时发到你最近在用的那个(修了一个「总挑最早绑的渠道」的隐藏 bug)。
- 👪 **家人名字不再认死** — 家人备注叫「蛋蛋(就是妈妈)」时,说「跟妈妈说一声」「发给蛋蛋」都能对上号;多人都对得上时它会如实报名单让你说清楚。
- 🖼️ **隔轮提图片也能接上** — 发完图过一两轮再说「把它下载/认一下」,它现在知道你刚发过图(以前它对上一轮的图完全失忆,会反过来找你要链接)。
- 💬 **微信会话有自己的图标** — 会话列表里微信对话不再是一个灰点。

### 修复

- 🔇 **根治「开关显示开、叫不答应,关掉重开才好使」** — 开机自启/冷启动时,语音引擎起来得比界面慢一步,界面从此认定「没开」、麦克风一直没开(v0.2.11 回声消除转默认后潜伏至今);现在引擎起来那一刻会主动通知界面开麦。重启机器后直接喊即可,不用再碰开关。
- 🔁 **改名后新喊法立即生效** — 以前在别的设置页改名,唤醒词根本没换(要去过「声音」页才生效);现在改完立刻能喊新名字。

> ⚠️ **待真机确认**:重启机器后直接喊「逼踢」应答(不碰设置);连着说长指令不截断;「发我微信」直达微信;放着歌喊它(回声消除开)不自我唤醒;开机自启藏托盘时若仍聋、但打开主窗后喊就好,单独反馈(那是另一个待修点)。

微信也接上了:扫码把它绑进你的微信,手机上直接聊;还学会「跟妈妈说一声…」——给家里人捎话、设提醒、发文件,话和文件都直达 TA 的手机。

### 新功能 / 改进

- 📨 **给家里人捎话 / 设提醒** — 「跟妈妈说一声,晚饭在锅里」「明早八点提醒爸爸吃药」:到点(捎话 = 马上)它会在 TA 手机的对话里开口转述,带着对 TA 的称呼和了解。TA 没连手机会当场如实告诉你,绝不设一个没人收的提醒;给家人设的提醒你和 TA 都看得见、都能反悔撤掉。
- 📤 **发文件也能指名** — 「把这份 PDF 发给爸爸」:文件直达 TA 手机上的对话(Telegram / 钉钉 / 微信都行)。
- 🪟 **窗口按平台各归其位** — Windows 保持无边框科幻窗,右上「方块」键改为最大化(铺满工作区、不再盖住任务栏,三键永远在、随时还原);Mac 改用原生标准窗(红绿灯,绿灯 = 系统真全屏)。沉浸式全屏专留给看视频。

### 修复

- 🖥️ **修「全屏后右上角按钮消失、退不出去」** — 之前手动把窗口放大到全屏后,右上三键会被误藏、又没有别的退出口,只能强杀进程;现在只有看视频的影院全屏才藏三键(那里有浮层 ✕ 和 Esc 可退)。
- 📄 **网页小窗点到文件链接不再干等** — 让它网页办事时,点到直接指向 PDF/附件的链接(Mac 上页面会变成文件查看器、读不出内容),现在它会把文件地址接给下载继续办,不再白等到超时说失败。
- 🖼️ **发图给不带视觉的模型,图不再白发** — 以前主脑不支持看图(如 DeepSeek)时,发图它只会说「看不了」再让你报文件路径;现在它知道自己有工具能直接处理你刚发的图(认二维码、转格式、读文档),对得上意图就直接办。

> ⚠️ **待真机确认**:微信全链(扫码 / 收发 / 图语音文件)需真微信号;捎话、跨人提醒、发文件到手机需真 Telegram / 真钉钉端到端;窗口改版两平台(Mac 红绿灯与真全屏、Windows 最大化不盖任务栏与卡死消失)均未真机验过,以真机为准。

## 0.2.13 — 2026-07-10

起什么名字就怎么喊它,喊了立刻「叮」一声;还学会一批办公小手艺——认二维码、开网页替你点按钮下文件、PDF 转图、把文件发到你手机。

### 新功能 / 改进

- 🏷️ **起什么名字,就怎么喊它** — 唤醒词不再单独设置:在「叫我什么」里改名,喊法自动跟着变(数字、字母都转成顺口的读法,比如 BT 喊「逼踢」、7274 喊「七二七四」)。默认名也换成了 **BT**。⚠️ 没改过名的家庭注意:老喊法「小七」退役了,这版起喊「逼踢」或「七二七四」。英文单词名(如 Buddy)喊不出来的话它会如实提示,先沿用默认喊法。
- 🔔 **喊它立刻「叮」一声** — 听到名字的瞬间先轻响一声,你第一时间知道它听见了(以前只有屏幕上的呼吸灯,不在电脑前经常不知道有没有喊到);是不是真在叫它、该不该答应,它接着在后台判断,误应答仍然很少。
- 🧰 **一批办公小手艺,一句话串着办** — 它现在会:认图里的二维码(手机拍了发它就行)、打开真网页替你点按钮(右下角小窗全程直播它在看哪页,要登录/扫码时你在小窗里完成、它接着办)、从网上下载文件、把 PDF 转成图片、把电脑上的文件发到你手机(Telegram / 钉钉)。像「把单据二维码里的链接打开、下载 PDF、转成图片发我手机」这样的组合活,交代一句就行。
- 💬 **一轮回答不再切成一串小气泡** — 它边查边说的回答现在合并在同一个气泡里,复制、朗读都是完整一段。
- 🧾 **设提醒、记事全程看得见** — 说完「提醒我…」,聊天里出现一张「已记下」小票(点一下直达提醒页);到点时聊天里先出一条「到点了」的线,然后它开口。让它「记住…」也有同款「记住了」小票。
- 🎛️ **「回声消除」开关转正** — 从试验块挪进设置·声音(麦克风选择旁):开着 = 放着歌、电影喊它更听得清(默认);不灵可一键切回经典采集。
- ⏳ **看片时喊它,音量压低得更短** — 说完话后它等你「继续说」的窗口在放片时从 6 秒缩到 3 秒,音量更快恢复。

### 修复

- 🔊 **音量不再「隔好几分钟才恢复」** — 喊它之后如果这轮失败、被打断或你切走了会话,以前媒体音量会卡在压低档很久;现在各种情况都及时恢复。
- ⏹️ **唤醒说话时,「停 / 定稿」按钮有反应了** — 以前这两个键只对打字框的听写有效,喊着说话时点了没用。
- 🖥️ **修开机自启后动画可能冻住** — 自启缩托盘后经悬浮窗打开主窗,桌宠和背景动画可能全部静止;现在会自己救活,各种打开方式都正常。
