# 更新日志

旺财 / Larkwing 的版本说明。版本号遵循语义化版本(SemVer);每个版本一节,标题格式 `## 版本号 — 日期`。

> 发布时,GitHub Release 的正文由 CI **自动抽取「对应版本这一节」**填入(见 `.github/workflows/release.yml`)。
> 出新版 = 在下方最上面加一节 `## x.y.z — 日期`,把该版变化写在它下面即可,无需手动填 Release 正文。

## 0.2.4 — 2026-07-03

家里人来了:每个人有自己的小本本;看电影声音也舒服了(不再小声、音量稳、深夜不吵人);自定义音色「不出声」查到根、修到位。

### 新功能

- 👨‍👩‍👧 **家里人来了(第一步)** — 设置·家人里可以把家里人一个个加进来;再把 TA 在 Telegram / 钉钉上跟旺财的对话「指认」给 TA。之后 TA 在手机上说「提醒我」「我爱吃什么」,旺财就知道这是 TA 自己的事:提醒发给 TA、喜好记在 TA 自己的小本本上,不会跟你的混一起。(用自己的声音让旺财认出是谁在说话,是下一步。)
- 🔊 **看电影声音不再特别小** — 有些片子(尤其国英双语 / 5.1 声道)转码后整体声音很小、得把系统音量开到吓人;现在电影、音乐会自动把音量拉稳、拉响,尖锐的峰值也不会突然炸耳朵。
- 🌙 **夜间模式** — 深夜看片自动把大动态压平些,不吵到家人。设置 ·「声音」可选 **关 / 开 / 自动**(自动默认 22:00–07:00,时段可改)。旺财说话时也会自动压低影视,先听清它。
- ✏️ **聊天自动起标题** — 新聊天不再一直叫「新会话」,聊上几句会自动起个简短的题目,列表里一眼认出;你自己改过的名字不会被覆盖。

### 修复

- 🎤 **自定义音色「点了不出声」** — 根因查实:声音模型下载时偶尔被截断,旧检查却当它是好的。现在模型下载带 sha256 与大小双重校验,坏文件自动识别并重新下载,不用你去翻文件夹;试听失败也会弹提示,不再悄无声息。
- 🐾 **开机自启后桌宠冻住** — 跟随开机启动再打开主界面时,桌宠有时卡成一张静止图、形象也切不了;现在打开窗口就正常动起来,换形象即时生效。

### 优化

- 🔈 **旺财说完话,电影声音缓缓回来** — 音量恢复从 0.7 秒放慢到 2.5 秒,不再突然轰回来。
- 🎚️ **音量均衡可关** — 「声音调稳」默认开;想要影院原汁原味的大动态,设置 ·「声音」里可关掉。

## 0.2.2 — 2026-06-30

接着打磨:发文件给旺财更顺手,国内更新更快,顺手修了个开机自启的烦人 bug。

### 新功能

- 📎 **拖文件、粘贴截图都能发了** — 现在可以把文件**直接拖进对话框**,或者截图后 **Ctrl+V 粘贴**进去,不用每次都点旁边的按钮。从文件管理器复制的文件,粘贴也能进来。
- 🎙️ **音色克隆能「现场录一段」** — 想让旺财用你的声音说话?在设置·声音里点「现场录一段」,对着麦克风说句话就行,不必再先录好音频文件再导入。

### 优化

- 🚀 **国内更新更快** — 查到新版本后,**下载安装包也走加速镜像**了(以前只有「检查」走镜像、下载还得连 GitHub),国内不开代理也更容易更新到。

### 修复

- 🖥️ **修好「开机自启后画面卡住」** — 跟着电脑开机启动、再打开主界面时,偶尔会只显示一个静止的形象、动画不动、点头像也不切换;现在恢复正常。

## 0.2.1 — 2026-06-30

接着 0.2.0 打磨:语音识别更准、追问文档更顺,给爱折腾的人加了模型「高级设置」。

### 优化

- 🎙️ **语音识别更准、不再出繁体** — 之前换识别模型那档可能把话识别成繁体、还不太准;现在精简成两档:**默认**(反应快)+ **更准一档**(听不清、或小朋友说话选这个,普通话更稳),都在本机识别。
- 📄 **接着问文档更顺** — 发一份文档问完再追问，文档内容还在，不用重新发一遍了。
- 🧠 **聊得久、读大文档更稳** — 旺财会按所用大脑的「记性大小」自动安排上下文，长对话和大文档都不容易卡。
- 🔄 **国内查更新更快** — 检查新版本时优先走加速镜像，够不到再走你的代理，更容易查到。
- 🔊 **语音过后音量恢复更柔和** — 喊它说话时会把正在放的影音压低，等它说完音量会**慢慢爬回来**，不再「啪」地一下轰回原音量，听感更自然。

### 新功能

- ⚙️ **模型「高级设置」(给爱折腾的人)** — 大脑设置里每个供应商卡片可以展开「高级」，手动校正某个模型的**智能档位 / 上下文长度 / 价格 / 计价方式**——填错或旺财没收录这个模型时用得上。一般人不用管，留空就按旺财自己的判断走。

## 0.2.0 — 2026-06-29

旺财这回学会了**看懂各种文档**,还能**动手帮你管电脑** —— 一个大版本。

### 新功能

- 📄 **看懂你的文档** — 把 Word、PPT、Excel、PDF 发给旺财(聊天里加文件,或直接说「看下 D:\账单.xlsx」),它就能读懂、帮你总结、念给你听。说明书、账单表、孩子的作业、PPT 都行;Excel 会按表格读、看得懂行列。(扫描成图片的 PDF 暂时读不出字。)
- 🖥️ **帮你管电脑** — 跟旺财说「锁屏」「睡一下」「把屏幕关了」「关机」「重启」,一句话搞定。关机和重启会留 **60 秒倒计时**,这期间改主意说一句「取消」就停下,不怕误关。
- ⌨️ **看片更顺手** — 放视频时:空格暂停 / 播放,上下方向键调音量,左右方向键快进、快退 20 秒。
- 💡 **更懂你下一步** — 比如正在放歌时,旺财会贴心给几条建议(「再来一首类似的」之类),点一下就发出去,不用自己想词。

### 改进

- 🛡️ **聊得久、读大文档也不卡住** — 优化了长对话和大文档的处理,不会再因为内容太长突然聊不动。
- 🔁 **下载失败能重试** — 组件下载没成功时,点一下「重试」就重来(以前只有放视频能重试)。
- 🔉 **放视频时调音量更靠谱** — 修了一个老问题:看片时喊它「大点声」,之前可能被唤醒避让(说话时自动压低)在收尾时还原音量给冲掉、白调一场;现在改动会保留生效。

## 0.1.8 — 2026-06-21

本地视频真放得了(含 5.1 声道 / mkv),看剧自动连播,语音识别可挑模型。

- 🔁 **B 站有时放不了修好了** — 开着全局代理时,B 站解析会被带去墙外节点而失败;现在固定走直连,不再受代理影响。

## 0.1.1 — 2026-06-17

旺财是面向普通人和家庭的桌面 AI 陪伴助手——说句话、点一下就能用,不用配置、不用懂技术。这是第一个版本。

### 能做什么

- 💬 **聊天陪伴** — 打字或语音都行,自然对话,会记住关于你和家里的事
- 🎬 **影音** — 一句话搜片、放片(B 站),还能用嘴控制播放 / 暂停 / 音量
- ⏰ **提醒待办** — 「明早八点提醒我吃药」随口说,到点来找你
- 🔎 **查资料** — 联网搜索直接带回要点,不用自己翻链接
- 📁 **管文件** — 整理 / 移动 / 改名 / 查找,删除走回收站、可撤销
- 🎙️ **语音唤醒** — 喊一声「小七」就应答(设置里可录音校准、改唤醒词)
- 🖥️ **常驻不打扰** — 开机自动上线、缩进系统托盘,悬浮窗随时唤回
- 📱 **手机也能聊** — 可接 Telegram / 钉钉机器人,人不在电脑前也能聊
- 🎨 **换肤** — 科幻 / 暖萌两套主题
- 🧠 **多模型接入** — 默认 DeepSeek,也支持 Gemini / OpenAI / 本地 Ollama;密钥存系统密钥串、不明文落库

### 安装

- **Windows(主要平台)** — 下载 `.exe` 安装。首次安装会默认开启「开机自启」,不想要可在 **设置 · 系统** 关掉,卸载会自动清理。
- **macOS** — 也提供 `.dmg`,供尝鲜。

### 说明

早期版本,语音、远程渠道等新能力仍在真机打磨中,遇到问题欢迎反馈。
