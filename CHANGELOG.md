# 更新日志

旺财 / Larkwing 的版本说明。版本号遵循语义化版本(SemVer);每个版本一节,标题格式 `## 版本号 — 日期`。

> 发布时,GitHub Release 的正文由 CI **自动抽取「对应版本这一节」**填入(见 `.github/workflows/release.yml`)。
> 出新版 = 在下方最上面加一节 `## x.y.z — 日期`,把该版变化写在它下面即可,无需手动填 Release 正文。

## 0.1.4 — 2026-06-19

新增护眼配色;放影音更"懂当下"、点播更准;记忆更会悟你的习惯。

### 新功能

- 🌿 **两套护眼配色** — 新增「豆沙绿(日间)」与「琥珀夜(夜间)」皮肤,长时间看更柔和(设置 → 外观)。

### 优化

- 🎬 **点播本地已有的影音更准** — 修正示范数据带来的误判,旺财不再"想当然"跳过你本地已有的电影 / 歌、直接上网找。
- 🎵 **更跟得上"现在在放什么"** — 一首歌 / 视频放完后,旺财不再以为还在播,聊起来更贴合当下。
- 🧠 **更会悟你的习惯** — 后台整理记忆时开了"思考",能从聊天里悟出你的长期偏好(比如"整理音乐按歌手"),同时该空就空、不乱记。

## 0.1.3 — 2026-06-18

记忆会自己长大、放视频更省心,数据还能搬到别的盘。

### 新功能

- 🧠 **自动记住重要的事** — 聊天时,旺财会把真正值得长期记住的悄悄记下来(没什么好记的就不记);在「记忆」页顶部可随时开关。
- 💾 **数据可以搬家** — 能把旺财的全部数据(聊天 / 记忆 / 下载的模型)整盘搬到别的磁盘,不用都堆在 C 盘(设置 → 系统)。
- 🖱️ **右键更顺手** — 聊天气泡、会话、记忆都能右键复制 / 删除;会话还能置顶、重命名。

### 优化

- 📺 **放需要登录的视频更省心** — 遇到要登录的内容不再直接报失败:扫码登录后会自动接着把它放出来,不用再说一遍。
- 🎙️ **更容易叫醒** — 默认把唤醒灵敏度调高了,喊「小七」更容易答应(觉得误触多可在设置里调低)。
- 🌐 **代理开关独立** — 代理的开关和地址分开了,临时关掉不丢地址,下次一开即用。

### 修复

- ✍️ **长内容不再被悄悄截断** — 之前写很长的备忘 / 记忆可能被截掉一截,现在会完整保留(实在太长会提示精简,不会闷头吃掉)。

## 0.1.2 — 2026-06-17

性能与体验打磨版——后台更省电、桌宠更安静。

### 优化

- ⚡ **更省 CPU、更省电** — 缩进系统托盘后,后台动画自动暂停(不再空耗 CPU);科幻主题背景降到 30fps 并去掉辉光描边,常开着也更轻快。
- 🐾 **桌宠更安静** — 旺财在聊天区漫步明显放慢,也不再推挤聊天气泡,陪着但不抢戏。

## 0.1.1 — 2026-06-17

旺财是面向普通人和家庭的桌面 AI 陪伴助手——说句话、点一下就能用,不用配置、不用懂技术。这是第一个版本。

### 能做什么

- 💬 **聊天陪伴** — 打字或语音都行,自然对话,会记住关于你和家里的事
- 🎬 **影音** — 一句话搜片、放片(B 站),还能用嘴控制播放 / 暂停 / 音量
- ⏰ **提醒待办** — 「明早八点提醒我吃药」随口说,到点来找你
- 🔎 **查资料** — 联网搜索直接带回要点,不用自己翻链接
- 📁 **管文件** — 整理 / 移动 / 改名 / 查找,删除走回收站、可撤销
- 🎙️ **语音唤醒** — 喊一声「小七」就应答(设置里可录音校准、改唤醒词)
- 🖥️ **常驻不打扰** — 开机自动上线、缩进系统托盘,悬浮窗随时唤回
- 📱 **手机也能聊** — 可接 Telegram / 钉钉机器人,人不在电脑前也能聊
- 🎨 **换肤** — 科幻 / 暖萌两套主题
- 🧠 **多模型接入** — 默认 DeepSeek,也支持 Gemini / OpenAI / 本地 Ollama;密钥存系统密钥串、不明文落库

### 安装

- **Windows(主要平台)** — 下载 `.exe` 安装。首次安装会默认开启「开机自启」,不想要可在 **设置 · 系统** 关掉,卸载会自动清理。
- **macOS** — 也提供 `.dmg`,供尝鲜。

### 说明

早期版本,语音、远程渠道等新能力仍在真机打磨中,遇到问题欢迎反馈。
